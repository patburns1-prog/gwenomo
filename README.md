# GWENOMO

Spanish-first interactive website project for Wendy's Gwenomo brand.

## Goal
Build a fun, visually striking, interactive website centered on handmade decorative gnomes, with a standout landing page and room to grow into a full storefront / catalog experience.

## Initial structure
- `site/` — frontend app
- `shared/` — handoff notes and assets for collaborators (Gemini / Claude / others)
- `assets/` — brand/source media copies for the project

## Current direction
- Language: predominantly Spanish
- Tone: warm, magical, handcrafted, festive
- Brand anchors: guardian gnomes, handmade with love, giftable, Ecuador origin, customizable colors/styles

## Collaboration notes
If external collaborators join, point them at `shared/PROJECT_BRIEF.md` first.
